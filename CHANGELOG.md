# Changelog

## [2.1.0] - 2026-06-19

### Added
- **Favicon**: Created custom SVG favicon with "PM" branding and gradient design
- **Social Media Meta Tags**: Added Open Graph and Twitter Card meta tags for better social sharing
- **Structured Data**: Implemented JSON-LD schema markup for enhanced SEO
- **Performance Optimizations**: Added preconnect hints for external resources
- **Deployment Script**: Created `deploy.sh` for easy deployment to multiple platforms
- **Placeholder Generator**: Added `create-placeholders.html` tool to generate professional gradient placeholders for project images
- **Enhanced SEO**: Improved meta descriptions and structured data

- **HTML Head Section**: Reorganized and optimized meta tags for better performance

- **Search Engine Visibility**: Enhanced with structured data markup
- **Developer Experience**: Added deployment automation script

### Technical Enhancements
