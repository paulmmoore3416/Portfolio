## 🚀 Quick Start

### View Live Site
Visit the live portfolio at: **[paulmmoore3416.github.io/Portfolio](https://paulmmoore3416.github.io/Portfolio)**

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/paulmmoore3416/Portfolio.git
   cd Portfolio
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   # Or use a local server:
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

3. **Make changes**
   - Edit `index.html` for content
   - Modify `styles.css` for styling
   - Update `script.js` for functionality
